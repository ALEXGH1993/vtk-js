title: MultiBlock
---

A MultiBlock is a set of other VTK data structures.

## Structure

```js
{
  type: 'MultiBlock',
  metadata: {
    name: 'can.ex2',
    size: 2345,
  },
  MultiBlock: {
    // FIXME
  },
}
```
